## **로직**

본래 제 풀이와는 다릅니다.

다른 분의 풀이를 가져와서 TDD를 적용해보았습니다.

1. map() 메서드로 배열에 접근한다.
2. map() 메서드 안에서 반복문을 돈다.
3. 현재 인덱스 값보다 인덱스 값이 작은 원소 중, 원소 값은 더 큰 값을 찾는다.
4. 값이 있다면 반복문을 빠져나오고 해당 인덱스 값에 +1을 더해 원소값을 변형한다.
5. 값이 없다면 0을 반환한다.
